# OMM–SBT: Oscillatory Mantle Model — Substantial Beating Theory

## Overview
This repository contains the reference implementation of the OMM–SBT framework, a causal-kinetic approach to emergent physical laws.

## Features
- Unified field evolution (X, Φ, K)
- Emergent geometry and flux structures
- Independent reproduction pipeline
- Invariance validation
- Generalized continuity analysis

## Installation
pip install -r requirements.txt

## Run
python experiments/run_simulation.py

## Reproduce key results
python validation/independent_reproduction.py

## Output
Results are exported as CSV files in `/results`.
